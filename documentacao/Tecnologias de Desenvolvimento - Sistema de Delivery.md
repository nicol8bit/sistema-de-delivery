# **Tecnologias de Desenvolvimento: Sistema de Delivery**

- **Linguagem de Programação:** PHP

- **Framework (Backend):**  Laravel 

- **Framework (Frontend):** HTML5, CSS3, JavaScript

- **Banco de Dados:**  MySQL

### **Justificativas das Escolhas**

- Maturidade e Comunidade: PHP é uma das linguagens mais maduras e amplamente utilizadas no desenvolvimento web, garantindo uma vasta comunidade, documentação e grande disponibilidade de profissionais.  
    
- Laravel: impõe uma arquitetura MVC (Model-View-Controller) limpa e organizada, facilitando a manutenibilidade e a escalabilidade do código. Laravel oferece recursos nativos para autenticação, rotas e segurança, acelerando o desenvolvimento.

- Universalidade: HTML5 e CSS3 são os padrões universais para a construção de interfaces web, garantindo portabilidade e compatibilidade com todos os navegadores modernos.

- Confiabilidade e Desempenho: MySQL é o SGBD relacional de código aberto mais popular, conhecido por sua confiabilidade, alta performance em leitura e escrita e facilidade de administração.

- Integração: MySQL possui integração nativa e otimizada com PHP e Laravel, formando uma stack coesa e de alto desempenho.
