# **Necessidades Tecnológicas: Sistema de Delivery**

#### **Ambiente de Execução e Dependências Mínimas:**

- Ambiente de Execução (Backend): PHP  
- Servidor Web: Apache  
- Banco de Dados: MySQL  
- Gerenciador de Dependências: Composer  
- Servidor Local: XAMPP  
- Dependências Mínimas (Backend): Laravel Framework, Extensões PHP  
- Dependências Mínimas (Frontend): HTML5, CSS3, JavaScript

#### **Requisitos de Hardware (Básicos para Desktop):**

- Processador: Dual-Core 2.0 GHz ou superior  
- Memória RAM: 8 GB  
- Armazenamento: 128 GB SSD  
- Sistema Operacional: Windows 10/11, macOS

#### **Requisitos de Hardware (Básicos para Celular):**

- Processador: Octa-Core 1.8 GHz ou superior / Apple A14 Bionic ou superior  
- Memória RAM: 4 GB  
- Sistema Operacional:  Android: versão 9 ou superior  /  iOS: versão 13 ou superior
